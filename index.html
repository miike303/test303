Gravity Lab

Simulation de gravité interactive

Par où commencer ?

  1. Clique sur Démarrer.
  2. Choisis un préréglage pour voir un style différent.
  3. Clique dans la zone de droite pour lancer une grosse rafale.
  4. Teste les curseurs pour voir l'effet immédiatement.

Tu n'as rien fait de mal : il manquait juste un vrai mode d'emploi au démarrage.

Astuce : commence par « Découverte », puis essaie « Tempête » ou « Lave ».
Que fait chaque option ?
  • Gravité : attire les particules vers le bas plus ou moins vite.
  • Débit : nombre de nouvelles particules ajoutées en continu.
  • Rebond : énergie gardée après un choc.
  • Frottement : ralentit progressivement le mouvement.
  • Vent : pousse les particules à gauche ou à droite.
  • Traînées : laisse un effet visuel de passage.
  • Attracteur souris : les particules suivent le curseur.
