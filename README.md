# Tetris Nova

Tetris Nova est une implémentation moderne de Tetris en HTML/CSS/JavaScript vanilla.

## Points clés

- Gameplay fidèle: déplacement, rotation, soft drop, hard drop, hold, queue "7-bag".
- Interface premium centrée sur la lisibilité du plateau.
- Panneau latéral clair: score, niveau, lignes, preview prochaine pièce, hold.
- Ghost piece, progression de niveau et vitesse dynamique.
- UI responsive avec écran de game over et bouton recommencer.

## Lancer

Ouvrez `index.html` dans un navigateur moderne, ou:

```bash
python3 -m http.server 8000
```

Puis visitez `http://localhost:8000`.

## Contrôles

- `←` `→` : déplacer
- `↑` ou `X` : rotation
- `↓` : descente rapide (soft drop)
- `Espace` : chute instantanée (hard drop)
- `C` ou `Shift` : hold
- `P` : pause
